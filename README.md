# Steam Shutdown Watcher

Самодостаточный BAT-скрипт для Windows: отслеживает загрузку и установку игры в Steam, затем выключает компьютер, когда отслеживаемая игра перестаёт изменяться.

Скрипт полезен, когда нужно оставить Steam докачивать игру и уйти. Для работы нужен только файл `SteamShutdownWatcher.bat`.

## Быстрый запуск

1. Скачайте `SteamShutdownWatcher.bat`.
2. Запустите его до начала загрузки или пока игра уже скачивается.
3. Оставьте окно скрипта открытым.

Скрипт сам находит Steam и библиотеки игр, определяет активный AppID и ждёт пять минут без изменений. После этого Windows получает команду выключения с задержкой 60 секунд.

## Безопасная проверка

Перед первым использованием можно убедиться, что скрипт правильно находит Steam, не выключая компьютер:

```bat
SteamShutdownWatcher.bat -DryRun
```

В этом режиме в окне и в логе появится строка о том, какая команда выключения была бы выполнена, но компьютер не выключится.

## Как отменить выключение

Пока идёт 60-секундный обратный отсчёт Windows, откройте `cmd` и выполните:

```bat
shutdown /a
```

Окно наблюдателя можно остановить в любой момент сочетанием `Ctrl+C` или закрытием окна.

## Параметры

| Параметр | Назначение | Пример |
| --- | --- | --- |
| `-DryRun` | Проверка без выключения ПК. | `SteamShutdownWatcher.bat -DryRun` |
| `-AppId <ID>` | Наблюдать за конкретной игрой, если её Steam AppID известен. | `SteamShutdownWatcher.bat -AppId 1194630` |
| `-SteamPath <путь>` | Указать папку Steam вручную, если поиск не сработал. | `SteamShutdownWatcher.bat -SteamPath "D:\\Steam"` |
| `-QuietMinutes <минуты>` | Время ожидания без изменений перед выключением. По умолчанию: `5`. | `SteamShutdownWatcher.bat -QuietMinutes 10` |
| `-PollSeconds <секунды>` | Интервал проверки Steam. По умолчанию: `15`. | `SteamShutdownWatcher.bat -PollSeconds 30` |
| `-ShutdownTimeout <секунды>` | Задержка Windows перед фактическим выключением. По умолчанию: `60`. | `SteamShutdownWatcher.bat -ShutdownTimeout 120` |

Несколько игр можно указать так:

```bat
SteamShutdownWatcher.bat -AppId 123456 -AppId 789012
```

## Как определяется загрузка

- Скрипт читает `appmanifest_*.acf` в библиотеках Steam и сразу замечает незавершённую загрузку или установку.
- Если Steam оставил старую папку в `steamapps\downloading`, она не считается активной, пока её содержимое не начнёт меняться между проверками.
- После обнаружения игры наблюдение ведётся только за её файлами и manifest, поэтому посторонняя активность Steam не должна сбрасывать таймер.

## Лог

Рядом с BAT-файлом создаётся `SteamShutdownWatcher.log`. В нём есть найденные библиотеки Steam, обнаруженные AppID, сбросы таймера и команда выключения. Если что-то работает не так, приложите этот файл к описанию проблемы.

## Требования

- Windows 10 или Windows 11;
- установленный Steam;
- встроенный Windows PowerShell 5.1 или новее.

## Важно

Запускайте скрипт до загрузки или во время неё. Если запускать его уже после полного завершения загрузки, в автоматическом режиме Steam может не предоставить признаков активной игры. В таком случае укажите AppID через `-AppId`.

Скрипт выключает Windows, поэтому перед запуском сохраните важную работу и не оставляйте незавершённые задачи в других программах.
